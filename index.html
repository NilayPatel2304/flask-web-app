Creating a Web Page with Flask

This page was created using the Flask framework. Below are the steps I followed to set up Flask and create this simple web page.

Installation Steps

  1. Created a Python virtual environment.
  2. Installed Flask using pip: pip install flask
  3. Created a simple Flask application in app.py.

Code Snippets

app.py

    from flask import Flask, render_template

    app = Flask(__name__)

    @app.route('/')
    def home():
        return render_template('index.html')

    if __name__ == '__main__':
        app.run(debug=True)

index.html

    <!DOCTYPE html>
    <html>
    <head>
        <title>My Flask App</title>
    </head>
    <body>
        <h1>Welcome to My Flask App!</h1>
    </body>
    </html>

Difficulties and Solutions

  • Difficulty: The Flask app didn't start due to a missing dependency.
  • Solution: Installed missing dependencies using pip install flask.

Conclusion

Using Flask was relatively simple for creating small web applications. I encountered a few minor issues but they were resolved by checking the Flask documentation.
